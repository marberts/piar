# Test if an object is an aggregation structure

Test if an object is a price index aggregation structure.

## Usage

``` r
is_aggregation_structure(x)
```

## Arguments

- x:

  An object to test.

## Value

Returns `TRUE` if `x` inherits from
[`piar_aggregation_structure`](https://marberts.github.io/piar/reference/aggregation_structure.md).
